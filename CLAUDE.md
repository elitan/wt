# Commits

- Use imperative verb: Add, Fix, Update, Remove, Refactor.
- Branch: `add-dark-mode`, `fix-login-bug`
- PR title / commit: `Add dark mode toggle`, `Fix login redirect bug`

# Release

- Check `git log` and merged PRs since last tag to determine semver bump (major requires user confirmation), then `git tag vX.Y.Z && git push --tags`.
